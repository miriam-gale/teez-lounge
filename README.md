# Teez Lounge - Restaurant Website

A modern, responsive restaurant website for Teez Lounge, a Ghanaian restaurant located in Accra, Ghana.

## Project Overview

This project was developed as a frontend web application showcasing the Teez Lounge brand, menu offerings, customer reviews, contact information, and online ordering experience.

The goal was to create a premium digital experience that reflects the restaurant's atmosphere while providing customers with easy access to menu items, reviews, and ordering options.

## Features

### Home Page
- Modern hero section
- Featured dishes
- Customer favourites
- About preview
- Reviews preview
- Contact section

### Menu Page
- Categorized menu items
- Food images
- Prices in Ghana Cedis (GHS)
- Search and filtering functionality
- Add-to-cart interface

### Reviews Page
- Customer testimonials
- Ratings display
- Social proof and trust indicators

### Contact Page
- Phone call integration
- WhatsApp integration
- Location information
- Contact details

### Responsive Design
- Mobile-first approach
- Tablet optimization
- Desktop optimization
- Smooth animations and transitions

## Technologies Used

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- JavaScript (ES6+)

## Development Approach

This project was built using a combination of traditional software development practices and AI-assisted development (vibe coding) through Claude Code.

AI tools were used to:
- Accelerate frontend development
- Generate component structures
- Improve UI/UX ideas
- Refactor and optimize code
- Speed up debugging and iteration

All implementation decisions, testing, customization, and project direction were managed by the developer.
